# Ordinary Differential Equations (ODEs) Study repository

This repository contains notes, exercises I made while studying ordinary differential equations (ODEs).

## Practical Exercises (Visualization & Engineering Applications)

These topics are suitable for coding tasks, simulations, and phase portrait analysis.

* **t1_separation_of_variables**: Implementation of basic growth/decay models.
* **t2_variation_of_constants**: Symbolic derivation of particular solutions (scalar & systems).
* **t3_bernoulli_equation**: Logistic growth and population dynamics.
* **t4_practical_ode_applications**: Modeling real-world systems (RC circuits, mixing problems).
* **t5_stability_of_equilibrium_1st_order**: Phase line analysis and stability of 1D systems.
* **t6_systems_of_equations**: Visualizing coupled systems.
* **t7_euler_method**: Numerical verification and error analysis.
* **t8_matrix_exponential**: Computation via Cayley-Hamilton theorem and minimal polynomials for system solutions.
* **t9_stability_of_equilibrium_points**: Classification of 2D linear systems (saddles, nodes, spirals).
* **t10_linear_system_flow**: Vector field visualization.
* **t11_linearization**: Using Jacobian analysis to approximate local behavior.
* **t12_local_behavior**: Phase portraits near fixed points.
* **t13_hamiltonian_systems**: Conservation laws and energy contours (e.g., pendulum).
* **t14_newtonian_systems**: Mechanical systems equations of motion.
* **t15_gradient_systems**: Potential functions and gradient descent flows.
* **t16_attractors**: Visualization of strange attractors and long-term behavior.
* **t17_periodic_cycles_and_stability**: Analyzing oscillating systems.
* **t18_poincare_map**: Discrete mapping of continuous flows.
* **t19_polar_coordinate_systems**: Transforming systems to polar coordinates to solve for cycles.
* **t20_limit_cycles**: Study of isolated closed orbits (Van der Pol oscillator).
* **t21_bifurcation_theory_basics**: Generating bifurcation diagrams and parameter sweeps.
